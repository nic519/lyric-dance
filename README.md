# Remotion Project

This is a Next.js template for building programmatic video apps with Remotion.

## Getting Started

Install dependencies:

```bash
bun install
```

## Commands

Start the Next.js dev server:

```bash
bun dev
```

Open the Remotion Studio:

```bash
bun remotion
```

Render a video locally:

```bash
bun render
```

Upgrade Remotion:

```bash
bun remotion upgrade
```

## Docs

Get started with Remotion by reading the [fundamentals page](https://www.remotion.dev/docs/the-fundamentals).
